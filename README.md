# 溯源 ODYSSEY 第42班 畢業紀念網站

這是一個溯源課程第42班畢業同學的紀念網站，參考了現代化的設計風格，並包含26位畢業同學的相框展示區域。

## 功能特色

- 🎨 現代化的設計風格，響應式布局
- 📸 26位畢業同學相框格子，支持點擊放大查看
- 📱 完全響應式設計，適配各種設備
- ✨ 平滑滾動和動畫效果
- 🖼️ 圖片模態框預覽功能
- 🎓 畢業紀念主題設計，包含畢業感言和旅程回顧

## 文件結構

```
obyssey42/
├── index.html      # 主頁面
├── styles.css      # 樣式文件
├── script.js       # JavaScript 功能
├── README.md       # 說明文件
└── images/         # 圖片目錄（需要創建）
    ├── person1.jpg
    ├── person2.jpg
    └── ... (共26張圖片)
```

## 使用說明

### 1. 準備圖片

在項目根目錄創建 `images` 文件夾，並放入26張畢業同學照片，命名為：
- `person1.jpg`
- `person2.jpg`
- `person3.jpg`
- ...
- `person26.jpg`

### 2. 打開網站

直接在瀏覽器中打開 `index.html` 文件即可查看網站。

### 3. 自定義內容

- 修改 `index.html` 中的文字內容
- 調整 `styles.css` 中的顏色和樣式
- 在 `script.js` 中修改功能邏輯

## 設計說明

### 顏色主題

- 主色調：深藍灰色 (#2c3e50)
- 強調色：紅色 (#e74c3c)
- 背景：漸變紫色（主視覺區域）

### 主要區域

1. **導航欄**：固定在頂部，包含導航連結
2. **主視覺區域**：全屏高度，展示第42班畢業紀念標題
3. **我們的旅程**：回顧六個月的探索與蛻變之旅
4. **畢業同學相框**：26位畢業同學的相框格子，網格布局
5. **畢業感言**：感謝與祝福的畢業感言
6. **頁尾**：版權信息

## 瀏覽器支持

- Chrome (最新版本)
- Firefox (最新版本)
- Safari (最新版本)
- Edge (最新版本)

## 注意事項

- 如果圖片文件不存在，相框會顯示數字佔位符
- 建議圖片尺寸為正方形（1:1比例）以獲得最佳顯示效果
- 圖片格式支持：JPG, PNG, WebP
