# 圖片目錄說明

請在此目錄中放置26張畢業同學照片。

## 圖片命名規則

請將圖片命名為：
- `person1.jpg`
- `person2.jpg`
- `person3.jpg`
- ...
- `person26.jpg`

## 圖片要求

- **格式**：JPG、PNG 或 WebP
- **尺寸**：建議使用正方形（1:1 比例），例如 800x800 像素
- **大小**：建議每張圖片不超過 500KB，以確保網站載入速度

## 注意事項

- 如果圖片不存在，網站會自動顯示數字佔位符
- 圖片會自動適應相框大小
- 點擊圖片可以放大查看

## 範例

如果您有26位畢業同學的照片，請按照以下方式命名：

```
images/
├── person1.jpg  (畢業同學1的照片)
├── person2.jpg  (畢業同學2的照片)
├── person3.jpg  (畢業同學3的照片)
├── ...
└── person26.jpg (畢業同學26的照片)
```
